Linux 入门很简单

Posted on 九月 3, 2019 本文总阅读量次

这是一本与众不同的 Linux 入门读物。作者借鉴历史畅销书《明朝那点事》的写作风格，将技术图书也以风趣幽默的风格一一道来，阅读起来十分过瘾。

书中以一个拟人化的 Ubuntu 操作系统为主角，以 Ubuntu10.04 为基础，讲解了Ubuntu 系统从安装、配置，到搭建开发平台、投入使用的过程。第 1 章介绍 Linux 系统的产生和发展概况；第 2 章介绍 Ubuntu 系统的各种安装方法；第 3 章介绍安装系统后的基本设置；第 4 章介绍 Ubuntu 下的常用软件；第 5 章介绍部分 Windows 下的软件在 Linux 中运行的解决方案；第 6 章介绍命令行的使用及脚本的编写；第 7 章介绍 Ubuntu 系统下 C/C++、Java、PHP 开发环境的搭建；第 8 章深入讲解一个软件的编译、调试、打包、发布的过程。
